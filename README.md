# Sweet Shop Management System

This repository contains a full-stack Sweet Shop Management System built with React (Vite), Node.js/Express, and MongoDB.

Folders:
- `backend` - Express API with Mongoose, JWT auth, and tests.
- `frontend` - Vite + React app with Tailwind UI and Axios for API calls.

Setup
1. Copy `.env.example` to `.env` in `backend` and fill Mongo URI and JWT secret.
2. From root run `npm run install-all` to install workspace dependencies.
3. Start both servers: `npm run dev` (from repo root).

Backend tests
- cd backend
- npm test
